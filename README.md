# py_chd_fuse

Mounting CHD files (Compressed Hunks of Data from MAME project) exposing BIN/CUE (track splitted)

## Usage:
```
python chd_fuse.py image.chd /mount/point/
```

## Requirements
- Linux (compatible with WSL), maybe *BSD and Mac OS?
- Python 3.8+
